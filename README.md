# Github-Repo-Explainer-Rag
RAG-powered AI assistant that explains GitHub repositories using semantic search, vector embeddings, and LLMs.

## Features
Multi-language repository analysis (8+ languages)
Tree-sitter based static code analysis
Folder hierarchy extraction
Function call graph generation
Import dependency graph generation
Semantic code chunking
ChromaDB vector database
Repository-aware conversational Q&A
Architecture summarization
